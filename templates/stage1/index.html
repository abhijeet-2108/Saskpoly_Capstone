🔍 Reconnaissance

🛰️ Nmap

Network scanning and port enumeration.
Nmap is a versatile tool that helps in discovering hosts and services on a computer network. It’s commonly used for security auditing and network inventory, providing detailed information about the network topology and potential vulnerabilities.

Launch Nmap
🌐 Whois

Fetch domain registration and ownership info.
Whois is a helpful tool for investigating domain names and IP addresses, providing contact and registration details. It’s essential for cybersecurity investigations, domain management, and verifying domain legitimacy.

Launch Whois
📡 DIG

DNS lookup and diagnostics utility.
DIG (Domain Information Groper) is widely used for DNS troubleshooting, providing detailed information about DNS records. It’s a must-have for network administrators and security professionals to ensure DNS configurations are correct and secure.

Launch DIG
🧭 NSLookup

Query DNS records and server information.
NSLookup is a command-line tool used to query DNS servers and obtain domain-related information. It’s crucial for diagnosing DNS issues, verifying records, and analyzing the resolution process.

Launch NSLookup
🕵️‍♂️ TheHarvester

Email, subdomain and open source intel gathering.
TheHarvester is an OSINT (Open Source Intelligence) tool that collects data from public sources like search engines and social networks. It’s valuable for ethical hacking, red teaming, and mapping an organization’s exposure on the internet.

Launch TheHarvester
